# 协作指南 - 大管家调用子机器人

## 团队架构

```
┌─────────────────────────────────────────────────┐
│              大管家 (主机器人)                    │
│   统筹策划 │ 项目管理 │ 质量检核                  │
└────────────────────┬────────────────────────────┘
                     │ 调度
       ┌───────┬─────┴─────┬───────┬───────┐
       ▼       ▼           ▼       ▼       ▼
   ┌───────┐┌───────┐┌───────┐┌───────┐
   │ 尔康  ││ 小马  ││ 小康  ││技术专家│
   └───────┘└───────┘└───────┘└───────┘
```

## 调用方式

```bash
# 调用尔康（策划运营）
openclaw agent --agent erkang --message "任务描述" --deliver

# 调用小马（产品研发）
openclaw agent --agent xiaoma --message "任务描述" --deliver

# 调用小康（数据分析）
openclaw agent --agent xiaokang --message "任务描述" --deliver

# 调用技术专家（防水技术）
openclaw agent --agent jishu-expert --message "任务描述" --deliver
```

## 团队成员

| 成员 | 专长 | 适合任务 |
|------|------|----------|
| **尔康** | 策划运营、市场调研 | 市场分析、运营策划、营销活动 |
| **小马** | 产品研发、自动化 | 产品开发、技术方案、自动化脚本 |
| **小康** | 数据分析 | 数据获取、趋势分析、定期报告 |
| **技术专家** | 防水技术 | 专业知识学习、技术咨询、案例积累 |

## 任务分配原则

### 尔康（策划运营）
- 市场调研和竞品分析
- 运营策略制定
- 营销活动策划
- 内容创意输出

### 小马（产品研发）
- 产品功能开发
- 技术方案设计
- 自动化脚本编写
- 系统集成

### 小康（数据分析）
- 定期数据获取
- 趋势分析和报告
- 异常监控
- 指标追踪

### 技术专家（防水技术）
- 防水专业知识学习
- 技术咨询和解答
- 案例积累和分析
- 知识库完善

## 协作流程

1. **接收任务** → 大管家接收用户任务
2. **任务分解** → 将复杂任务分解为子任务
3. **分配任务** → 根据专长分配给对应成员
4. **监控进度** → 跟踪任务执行状态
5. **质量检核** → 审核子机器人产出
6. **汇总汇报** → 向用户汇报最终结果
