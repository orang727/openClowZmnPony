## 📋 任务 2：施工方案映射表（✅ 完成）

### 核心流程框架（含实地勘察）

```
用户提交 → AI 初步分析 → 师傅实地勘察 → 确认/修正 → 生成施工方案 → 报价 → 用户确认 → 施工
           ↓              ↓
      面积/病害/严重   必查项：
      程度识别        - 位置确认
                      - 面积测量
                      - 病害类型
                      - 严重程度
                      - 施工难度
                      - 隐藏问题检测
```

### 映射表结构（5 个维度）

| 维度 | 取值 | 数量 |
|------|------|------|
| 漏点位置 | 卫生间/厨房/阳台/屋顶/外墙/飘窗/其他 | 7 |
| 面积区间 | 小 (<5㎡)/中 (5-15㎡)/大 (>15㎡) | 3 |
| 病害类型 | 裂缝/空鼓/渗水 | 3 |
| 严重程度 | 轻微/普通/严重 | 3 |
| **理论场景数** | **7×3×3×3 = 189 种** |
| **简化后** | **38 个标准场景** |

## 📋 任务 3：价格模型实现方案（✅ 完成）

### 核心价格公式

```
最终价格 = 基础价格 × 位置系数 × 难度系数 × 渗漏系数 + 附加费用
```

### 系数表（核心）

| 系数类型 | 取值范围 | 说明 |
|---------|---------|------|
| 位置系数 | 0.9-1.3 | 卫生间 1.0/屋顶 1.2/外墙 1.3 |
| 难度系数 | 0.8-1.8 | 轻微 0.8/普通 1.0/严重 1.5 |
| 渗漏系数 | 0.8-1.5 | 轻微 0.8/普通 1.0/严重 1.5 |
| 附加费用 | 0-2000 元 | 高空作业/紧急加急等 |

### 数据字段需求清单（30 个核心字段）

**给泊宁导表 - 必填字段**：

| 序号 | 字段名 | 说明 | 示例 |
|------|--------|------|------|
| 1 | order_id | 工单 ID | ORD20250101001 |
| 2 | location_type | 漏点位置 | 卫生间 |
| 3 | area | 面积（㎡） | 6.5 |
| 4 | damage_type | 病害类型 | 裂缝 |
| 5 | severity | 严重程度 | 普通 |
| 6 | construction_plan | 施工方案 | 局部注浆 + 表面防水 |
| 7 | base_price | 基础价格 | 1000 |
| 8 | material_cost | 材料费 | 400 |
| 9 | labor_cost | 人工费 | 600 |
| 10 | total_price | 总价 | 1000 |
| 11 | actual_price | 实际成交价 | 980 |
| 12-30 | ... | 详见详细文档 | ... |

**数据时间范围**: 2000-01-01 至 2026-03-01  
**交付格式**: CSV 或 Excel
