## 🎯 核心信息提取

### 1. 产品定位（小马版）

> 用户拍照即可获得防水故障评估、施工方案推荐和价格区间，快速生成工单的 H5 应用，嵌入**微信/抖音小程序 WebView**。

**当前变更**: 改为嵌入**官网品牌站**

### 2. 用户流程（6 步）

```
进入 H5 → 填写表单 → 上传图片 → 点击估价 → 展示结果 → 选择方案 → 确认下单 → 返回小程序
```

**当前流程**（简化版）:
```
P1 选择位置 → P2 上传图片 → P3 AI 分析 → P4 方案推荐 → P5 价格显示 → P6 确认下单
```

| 序号 | 功能模块 | 优先级 | 状态 |
|------|---------|--------|------|

| 2 | 建筑信息填写（可选） | P1 | ✅ 已定义 |
| 3 | 问题描述（200 字） | P1 | ✅ 已定义 |

### 4. AI 识别病害类型（6 种）

| 病害类型 | 说明 | 典型场景 |
|---------|------|---------|
| 水渍 | 墙面/地面水渍印记 | 卫生间墙角发霉变黑 |
| 裂缝 | 墙面/地面裂缝 | 屋顶裂纹、墙体开裂 |
| 潮湿 | 局部潮湿、凝水 | 地下室墙面渗水 |
| 泛碱 | 白色碱化结晶 | 瓷砖缝、外墙起白硝 |
| 渗漏 | 明显漏水痕迹 | 水珠滴落、明显水流 |
| 发霉 | 霉菌斑块 | 黑色/绿色霉斑 |

### 5. 严重程度判定

| 等级 | 判定依据 |
|------|---------|
| **轻微** | 局部小面积（<10cm²）、早期症状、不影响结构 |
| **普通** | 明显可见、范围适中（10-50cm²）、需要处理 |
| **严重** | 大面积（>50cm²）、已影响结构、需紧急处理 |

### 6. 故障 - 方案映射（核心 10 种场景）

| 漏点位置 | 故障类型 | 严重程度 | 方案名称 | 工期 | 价格区间 |
|---------|---------|---------|---------|------|---------|
| 卫生间 | 水渍 | 轻微 | 局部防水涂层 | 0.5 天 | 200-500 元 |
| 卫生间 | 水渍 | 普通 | 卫生间重做防水 | 2-3 天 | 1500-3000 元 |
| 卫生间 | 裂缝 | 轻微 | 裂缝灌浆处理 | 0.5 天 | 300-600 元 |
| 卫生间 | 渗漏 | 严重 | 注浆防水 + 整体重做 | 3-5 天 | 3000-6000 元 |
| 屋顶 | 裂缝 | 普通 | 裂缝修补 + 防水卷材 | 1-2 天 | 1000-2000 元 |
| 屋顶 | 渗漏 | 严重 | 整体防水 + 隔热层 | 5-7 天 | 5000-10000 元 |

**⚠️ 待确认**: 这些方案和价格需要业务专家评审！
