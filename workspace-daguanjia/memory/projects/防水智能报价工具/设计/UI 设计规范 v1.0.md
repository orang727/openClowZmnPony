## 📱 响应式规范

### 断点

| 设备 | 宽度 | 说明 |
|------|------|------|
| 手机 | <768px | 单栏布局 |
| 平板 | 768px-1024px | 双栏布局 |
| 桌面 | >1024px | 多栏布局 |

### 适配原则

1. **移动优先** - 先设计手机版，再适配大屏
2. **弹性布局** - 使用百分比和 flex 布局
3. **图片适配** - 使用 srcset 或响应式图片

---

## 🎯 设计原则

### 视觉原则

1. **层次清晰** - 重要信息突出，次要信息弱化
2. **对比适度** - 颜色、大小、粗细对比合理
3. **对齐规范** - 严格遵循栅格和对齐规则
4. **留白呼吸** - 给内容足够的呼吸空间

### 交互原则

1. **反馈及时** - 操作后有明确反馈
2. **状态清晰** - 各种状态（默认/hover/active/disabled）明确
3. **容错设计** - 允许用户犯错，提供撤销
4. **引导明确** - 视觉引导用户下一步操作
