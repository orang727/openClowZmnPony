## 7 大异常场景

### 1. 网络超时

```
┌─────────────────────────────┐
│  ←  返回                    │
├─────────────────────────────┤
│                             │
│        ⏱️                  │
│      (80px)                 │
│                             │
│   网络连接超时              │
│                             │
│   网络好像有点慢            │
│   请检查网络设置后重试       │
│                             │
│   可能原因：                 │
│   • 网络信号弱              │
│   • WiFi 连接不稳定          │
│   • 移动数据未开启          │
│                             │
├─────────────────────────────┤
│  ┌─────────────────────┐   │
│  │    重新尝试         │   │
│  └─────────────────────┘   │
│                             │
│  [ 返回上一步 ]             │
│                             │
└─────────────────────────────┘
```

**触发条件**: 请求超过 15 秒无响应  
**系统处理**: 保留用户已输入数据，支持重试

## 设计规范

### 图标规范

| 场景 | 图标 | 尺寸 | 颜色 |
|------|------|------|------|
| 网络超时 | ⏱️ | 80px | #1E88E5 |
| AI 失败 | 🤖 | 80px | #FB8C00 |
| 上传失败 | 📷 | 80px | #E53935 |
| 面积失败 | 📏 | 80px | #1E88E5 |
| 价格失败 | 💰 | 80px | #FB8C00 |
| 提交失败 | ⚠️ | 80px | #E53935 |
| 加载失败 | 📄 | 80px | #999999 |
| 成功 | ✅ | 64px | #43A047 |

### 文字规范

| 元素 | 字号 | 字重 | 颜色 |
|------|------|------|------|
| 错误标题 | 20px | Medium | #333333 |
| 错误描述 | 14px | Regular | #666666 |
| 原因列表 | 13px | Regular | #999999 |
| 按钮文字 | 16px | Medium | #FFFFFF |
| 次要操作 | 14px | Regular | #1E88E5 |

### 间距规范

| 元素 | 间距 |
|------|------|
| 图标到标题 | 24px |
| 标题到描述 | 12px |
| 描述到原因 | 24px |
| 原因到按钮 | 48px |
| 主按钮到次要 | 16px |

## 使用指南

### 何时使用通用错误页

- ✅ 网络请求失败
- ✅ 服务不可用
- ✅ 数据加载失败
- ✅ 操作执行失败

### 何时使用 Toast

- ✅ 轻量级提示
- ✅ 操作成功反馈
- ✅ 状态变更通知
- ❌ 严重错误（用错误页）

### 何时使用 Loading

- ✅ 耗时操作（>1 秒）
- ✅ 数据加载中
- ✅ 提交处理中
- ❌ 瞬时操作（<0.5 秒）
