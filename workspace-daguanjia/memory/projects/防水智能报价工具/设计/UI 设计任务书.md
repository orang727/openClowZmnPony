## 📋 任务概述

### 项目背景
- **项目名称**: 防水智能报价工具
- **产品形态**: H5 页面（嵌入官网品牌站）
- **核心价值**: 用户拍照→AI 分析→施工方案→价格区间→生成工单
- **目标上线**: 2026-03-31
- **当前阶段**: Gate 5 - UI 设计（加急）

### 设计范围

- **设计端**: H5 端（优先）
- **设计风格**: 专业、可信赖、简洁

## 🎨 设计要求

### 风格定位
| 维度 | 要求 |
|------|------|
| **整体风格** | 专业、可信赖、简洁 |
| **色彩倾向** | 蓝色系（专业、技术感） |
| **视觉感受** | 清晰、易读、操作流畅 |
| **品牌调性** | 啄木鸟防水 - 专业防水服务 |

### 设计原则
1. **用户第一** - 减少认知负担，让用户少思考
2. **简单清晰** - 信息层次清晰，操作直观
3. **建立信任** - 专业感、可信度、透明度
4. **引导转化** - 视觉引导到下一步，提升转化率

## 📐 页面详细需求

### P1. 漏点位置选择

**页面结构**:
```
┌─────────────────────────┐
│   请选择漏点位置         │ ← 页面标题
├─────────────────────────┤
│  ┌───────┐ ┌───────┐   │
│  │ 🚽    │ │ 🍳    │   │ ← 位置卡片（带图标）
│  │卫生间  │ │ 厨房   │   │
│  └───────┘ └───────┘   │
│  ┌───────┐ ┌───────┐   │
│  │ 🌤️    │ │ 🏠    │   │
│  │ 阳台   │ │ 屋顶   │   │
│  └───────┘ └───────┘   │
│  ... (共 7 个选项)       │
├─────────────────────────┤
│      [下一步]           │ ← 底部固定按钮
└─────────────────────────┘
```

**设计要点**:
- 卡片尺寸：160x120px
- 图标尺寸：48x48px
- 选中状态：边框高亮 + 背景色
- 按钮：固定在底部

---

### P2. 上传现场图片

**页面结构**:
```
┌─────────────────────────┐
│   上传现场照片           │ ← 页面标题
│   上传 1-5 张照片         │ ← 副标题
├─────────────────────────┤
│  ┌─────────────────┐   │
│  │    📷           │   │ ← 上传区域
│  │ 点击拍照或       │   │
│  │ 从相册选择       │   │
│  └─────────────────┘   │
│  ┌───┐ ┌───┐ ┌───┐   │ ← 已上传缩略图
│  │ × │ │ × │ │ × │   │
│  └───┘ └───┘ └───┘   │
│  已上传 3/5 张         │ ← 数量提示
├─────────────────────────┤
│ [上一步]    [下一步]   │
└─────────────────────────┘
```

**设计要点**:
- 上传区域：200x200px，虚线边框
- 缩略图：80x80px，可删除
- 支持：拍照、相册选择

### P3. AI 分析中

**页面结构**:
```
┌─────────────────────────┐
│   AI 智能分析中          │ ← 页面标题
├─────────────────────────┤
│      ⚙️🔄⚙️            │ ← 加载动画
│      (动态效果)          │
├─────────────────────────┤
│  ✓ 正在识别漏水位置...   │ ← 进度提示
│  ○ 正在分析严重程度...   │
│  ○ 正在匹配施工方案...   │
├─────────────────────────┤
│   预计需要 10 秒          │
└─────────────────────────┘
```

**设计要点**:
- 动画：科技感、专业感
- 进度：分步显示，缓解焦虑
- 时间：明确告知等待时间

## 🎨 设计规范

### 色彩方案（建议）

| 类型 | 色值 | 用途 |
|------|------|------|
| **主色** | #1E88E5 | 品牌蓝，按钮、链接 |
| **成功色** | #43A047 | 成功状态 |
| **警告色** | #FB8C00 | 警告提示 |
| **错误色** | #E53935 | 错误状态 |
| **信息色** | #1E88E5 | 信息提示 |
| **文字主色** | #333333 | 主要文字 |
| **文字次要** | #666666 | 次要文字 |
| **文字提示** | #999999 | 提示文字 |
| **背景色** | #F5F5F5 | 页面背景 |
| **卡片背景** | #FFFFFF | 卡片背景 |

### 字体方案

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| 标题 | 24px | Bold | 1.4 | 页面标题 |
| 副标题 | 18px | Medium | 1.4 | 模块标题 |
| 正文 | 16px | Regular | 1.6 | 主要内容 |
| 辅助文字 | 14px | Regular | 1.5 | 说明文字 |
| 提示文字 | 12px | Regular | 1.5 | 次要提示 |
