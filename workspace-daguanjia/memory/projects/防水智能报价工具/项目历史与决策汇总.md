## 📊 项目方案详情（PRD v2.0 核心）

### 产品定位
- **名称**: 防水补漏智能报价器
- **形态**: H5 页面（嵌入官网品牌站）
- **核心价值**: 用户拍照→AI 分析→施工方案→价格区间→生成工单

### P0 功能（必须）
1. 漏点位置选择（卫生间/厨房/阳台/屋顶/外墙/飘窗/其他）
2. 现场图片上传（1-5 张）
3. AI 图像分析（识别严重程度：轻微/普通/严重）
4. 智能方案推荐（根据严重程度匹配施工方案）
5. 价格区间显示
6. 确认下单
7. 工单推送

### 6 步用户流程
1. **选择漏点位置** → 2. **上传现场图片** → 3. **AI 智能分析** → 4. **推荐施工方案** → 5. **显示价格区间** → 6. **确认下单**

### 技术架构
- **前端**: H5（响应式设计）
- **AI 分析**: 调用 AI 模型 API 识别漏水严重程度
- **图片上传**: 前端压缩后上传，最大 8MB，支持 jpg/png
- **工单推送**: 推送到现有工单系统
- **部署**: AI 开发 + 线下员工部署测试

### 性能要求
- 页面加载 < 3 秒
- 图片上传 < 5 秒
- 估价响应 < 10 秒（含 AI 分析）
