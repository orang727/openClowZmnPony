## 价格公式总览

### 核心公式

```
最终价格 = (基础材料费 + 人工费 + 措施费) × 位置系数 × 难度系数 + 附加费用
```

### 公式说明

| 组成部分 | 说明 | 是否必选 |
|---------|------|---------|
| 基础材料费 | 防水材料费用 | ✅ 必选 |
| 人工费 | 施工人员工资 | ✅ 必选 |
| 措施费 | 成品保护、垃圾清运等 | ✅ 必选 |
| 位置系数 | 不同施工位置的难度调整 | ✅ 必选 |
| 难度系数 | 病害严重程度调整 | ✅ 必选 |
| 附加费用 | 紧急施工、特殊工艺等 | ⚠️ 可选 |

### 2. 人工费

```
人工费 = 工日 × 日工资
```

|------|------|---------------|------|
| **防水工** | 普通 | 300-400 | 常规防水施工 |
| **防水工** | 熟练 | 400-600 | 复杂工艺、高空作业 |
| **防水工** | 专家 | 600-800 | 疑难问题、技术指导 |
| **小工** | 普通 | 150-250 | 辅助工作、搬运 |
| **高空作业** | 特种 | +200-300/天 | 外墙/屋顶高空补贴 |
| **夜间施工** | 加班 | +50% | 18:00-次日 6:00 |

| **管根渗漏** | 1.0 | 1.2 | 1.8 |
| **接缝渗漏** | 1.0 | 1.2 | 1.6 |
| **表面渗水** | 1.0 | 1.3 | 1.8 |

#### 难度判定规则

```
轻微：面积小、表观问题、简单修复
普通：面积中等、需专业处理、标准工艺
严重：面积大、影响结构、复杂工艺、多次施工
```

## 价格上限与审核机制

### 价格上限

| 类型 | 上限金额 | 说明 |
|------|---------|------|
| **AI 自动报价上限** | 5000 元 | 超过需人工审核 |
| **单次报价浮动范围** | ±30% | 相对于基准价 |
| **最高报价** | 20000 元 | 需业务专家审批 |

### 人工审核触发条件

出现以下任一情况，转人工审核：

1. **报价超过 5000 元**
2. **病害类型为"严重"且面积>10㎡**
3. **涉及结构安全（结构性裂缝）**
4. **用户要求特殊工艺或材料**
5. **历史投诉高风险小区**
6. **价格公式计算结果异常（偏离基准价>50%）**

### 审核流程

```
AI 估价 → 触发审核条件？
├─ 否 → 直接展示给用户
└─ 是 → 转人工审核 → 业务专家确认 → 展示给用户
```

## 价格计算示例

### 示例 1：卫生间管根渗漏（普通）

```
基础材料费：堵漏王 2 袋 × 60 元 + 防水涂料 1 桶 × 200 元 = 320 元
人工费：1 工日 × 400 元 = 400 元
措施费：成品保护 100 元 + 垃圾清运 100 元 = 200 元
小计：320 + 400 + 200 = 920 元

位置系数：卫生间 = 1.0
难度系数：管根渗漏（普通）= 1.2

最终价格 = 920 × 1.0 × 1.2 = 1104 元
价格区间：1104 × (0.7~1.3) = 773~1435 元
```

### 示例 2：屋顶裂缝（严重）

```
基础材料费：灌浆材料 5 支 × 50 元 + 防水卷材 30㎡ × 35 元 = 1300 元
人工费：3 工日 × 500 元（高空补贴）= 1500 元
措施费：垃圾清运 400 元 + 成品保护 200 元 = 600 元
小计：1300 + 1500 + 600 = 3400 元

位置系数：屋顶 = 1.2
难度系数：裂缝（严重）= 1.5

最终价格 = 3400 × 1.2 × 1.5 = 6120 元
→ 超过 5000 元，触发人工审核
```
