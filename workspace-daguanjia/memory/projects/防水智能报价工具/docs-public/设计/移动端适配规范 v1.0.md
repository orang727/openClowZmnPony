### 兼容性处理策略

#### iOS Safari 特别处理
```css
/* 安全区域适配（iPhone X+） */
padding: env(safe-area-inset-top) env(safe-area-inset-right) 
         env(safe-area-inset-bottom) env(safe-area-inset-left);

/* 禁止缩放 */
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">

/* 禁止长按选中 */
-webkit-touch-callout: none;
-webkit-user-select: none;
```

#### 微信内置浏览器特别处理
```javascript
// 微信 JSSDK 配置（如需分享等功能）

// 检测微信环境

```

#### Android 兼容性处理
```css

@media (-webkit-min-device-pixel-ratio: 2) {
  .border-1px {
    transform: scaleY(0.5);
  }
}

/* 修复 Android 下 input 焦点样式 */
input:focus, textarea:focus {
  outline: none;
  -webkit-tap-highlight-color: transparent;
}
```

### 屏幕适配

#### 适配设备尺寸

| 设备类型 | 屏幕宽度 | 代表设备 | 适配方案 |
|---------|---------|---------|---------|
| **小屏手机** | 320-360px | iPhone SE, 小屏 Android | 单列布局，精简内容 |
| **标准手机** | 375-414px | iPhone 6-11, 主流 Android | 标准布局 |
| **大屏手机** | 414-480px | iPhone Pro Max, 大屏 Android | 适度放宽内容区 |
| **小屏平板** | 600-768px | iPad mini, 小平板 | 双列布局 |
| **标准平板** | 768-1024px | iPad, 标准平板 | 双列/三列布局 |
| **大屏平板** | 1024px+ | iPad Pro, 大平板 | 多列布局 |

#### 响应式断点

```css
/* 移动优先 */
/* 默认样式：手机竖屏 */

/* 大屏手机 */
@media (min-width: 414px) {
  .container { padding: 20px; }
}

/* 小屏平板 */
@media (min-width: 600px) {
  .container { max-width: 560px; margin: 0 auto; }
}

/* 标准平板 */
@media (min-width: 768px) {
  .container { max-width: 720px; }
  .card-grid { grid-template-columns: repeat(2, 1fr); }
}

/* 大屏平板/桌面 */
@media (min-width: 1024px) {
  .container { max-width: 960px; }
  .card-grid { grid-template-columns: repeat(3, 1fr); }
}
```

#### 核心适配方案

**1. 弹性布局（Flexbox）**
```css
.container {
  display: flex;
  flex-direction: column;
  padding: 16px;
}

.card {
  flex: 1;
  min-width: 0; /* 防止内容溢出 */
}
```

**2. 栅格系统**
```css
/* 12 栏栅格 */
.grid {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  gap: 16px;
}

.col-12 { grid-column: span 12; }
.col-6 { grid-column: span 6; }
.col-4 { grid-column: span 4; }

/* 移动端默认全宽 */
@media (max-width: 767px) {
  .col-6, .col-4 { grid-column: span 12; }
}
```

**3. 相对单位**
```css
/* 使用 rem 适配字体 */
html { font-size: 16px; }
@media (max-width: 375px) { html { font-size: 14px; } }
@media (min-width: 768px) { html { font-size: 18px; } }

/* 使用 vw/vh 适配容器 */
.container {
  width: 100%;
  max-width: 100vw;
  overflow-x: hidden;
}
```

**4. 图片适配**
```css
img {
  max-width: 100%;
  height: auto;
  display: block;
}

/* 响应式图片 */
<picture>
  <source media="(min-width: 768px)" srcset="image-large.jpg">
  <source media="(min-width: 375px)" srcset="image-medium.jpg">
  <img src="image-small.jpg" alt="...">
</picture>
```

### 深色模式支持

#### 支持策略

| 模式 | 说明 | 实现方式 |
|------|------|---------|
| **跟随系统** | 默认策略，自动适配 | prefers-color-scheme |
| **手动切换** | 用户可手动切换 | 本地存储偏好 |
| **强制浅色** | 品牌色不支持深色时 | 固定浅色主题 |

#### 深色模式配色

```css
/* 浅色模式（默认） */
:root {
  --bg-primary: #FFFFFF;
  --bg-secondary: #F5F5F5;
  --text-primary: #333333;
  --text-secondary: #666666;
  --text-hint: #999999;
  --border: #CCCCCC;
  --primary: #1E88E5;
}

/* 深色模式 */
@media (prefers-color-scheme: dark) {
  :root {
    --bg-primary: #1A1A1A;
    --bg-secondary: #2D2D2D;
    --text-primary: #FFFFFF;
    --text-secondary: #CCCCCC;
    --text-hint: #999999;
    --border: #404040;
    --primary: #42A5F5; /* 调亮主色 */
  }
}
```

#### 深色模式适配清单

| 元素 | 浅色模式 | 深色模式 | 注意事项 |
|------|---------|---------|---------|
| 页面背景 | #F5F5F5 | #1A1A1A | 避免纯黑#000000 |
| 卡片背景 | #FFFFFF | #2D2D2D | 保持对比度 |
| 主色按钮 | #1E88E5 | #42A5F5 | 深色下调亮 |
| 文字主色 | #333333 | #FFFFFF | 保证可读性 |
| 文字次要 | #666666 | #CCCCCC | - |
| 边框 | #CCCCCC | #404040 | 降低对比度 |
| 阴影 | rgba(0,0,0,0.12) | rgba(0,0,0,0.4) | 加深阴影 |
| 图片 | 原色 | 原色（或微调） | 避免过曝 |

#### 深色模式测试项

- [ ] 所有页面在深色模式下可读
- [ ] 对比度符合 WCAG AA 标准（≥4.5:1）
- [ ] 图片/图标在深色背景下清晰
- [ ] 输入框、按钮状态明确
- [ ] 弹窗、浮层背景区分明显

### 单手操作优化

#### 热区划分

```
┌─────────────────────────────┐
│  难以触及区 (顶部 20%)       │  ← 放置次要操作
│  ← 返回 / 标题              │
├─────────────────────────────┤
│                             │
│  舒适操作区 (中部 60%)       │  ← 核心内容区
│  • 卡片列表                 │
│  • 表单输入                 │
│  • 图片上传                 │
│                             │
├─────────────────────────────┤
│  易触及区 (底部 20%)         │  ← 主要操作按钮
│  [ 上一步 ]  [ 下一步 ]     │
└─────────────────────────────┘
```

#### 按钮尺寸规范

| 按钮类型 | 最小尺寸 | 推荐尺寸 | 说明 |
|---------|---------|---------|------|
| **主要按钮** | 48×48px | 48px 高（全宽） | 底部固定，拇指易触 |
| **次要按钮** | 44×44px | 40px 高 | 常规操作 |
| **图标按钮** | 44×44px | 48×48px | 点击区域≥44px |
| **列表项** | 44px 高 | 56px 高 | 行高足够 |

#### 单手操作设计原则

**1. 核心操作底部化**
```
❌ 错误：主要按钮在顶部
┌─────────────────┐
│  [ 提交订单 ]   │  ← 难以触及
│                 │
│  ...内容...     │
└─────────────────┘

✅ 正确：主要按钮在底部
┌─────────────────┐
│  ...内容...     │
│                 │
│  [ 提交订单 ]   │  ← 拇指易触
└─────────────────┘
```

**2. 返回操作左上角**
- 返回按钮固定在左上角（符合用户预期）
- 支持手势返回（iOS 左滑）
- 返回按钮点击区域≥44×44px

**3. 表单输入优化**
```css
/* 输入框高度足够 */
input, textarea {
  min-height: 48px;
  font-size: 16px; /* 防止 iOS 自动缩放 */
  padding: 12px 16px;
}

/* 焦点状态明显 */
input:focus {
  border-color: #1E88E5;
  box-shadow: 0 0 0 2px rgba(30,136,229,0.2);
}
```

**4. 滚动优化**
```css
/* 平滑滚动 */
html {
  scroll-behavior: smooth;
}

/* 滚动条隐藏（美观） */
::-webkit-scrollbar {
  display: none;
}

/* 防止滚动穿透 */
body.modal-open {
  position: fixed;
  overflow: hidden;
}
```

**5. 手势支持**
```javascript
// 左滑返回（iOS 风格）
import { useSwipe } from './hooks';

useSwipe({
  onSwipeLeft: () => history.back(),
  threshold: 50, // 滑动距离阈值
});

// 下拉刷新
usePullToRefresh({
  onRefresh: () => loadData(),
  threshold: 80,
});
```

### 性能优化

#### 加载性能

| 指标 | 目标值 | 优化方案 |
|------|--------|---------|
| 首屏加载 | <3 秒 | 资源压缩、CDN、懒加载 |
| 图片上传 | <5 秒 | 压缩、分片上传、进度提示 |
| 交互响应 | <100ms | 防抖、节流、Web Worker |
| 页面切换 | <300ms | 预加载、缓存 |

#### 图片优化
```javascript
// 上传前压缩
const compressImage = (file, quality = 0.7) => {
  return new Promise((resolve) => {
    const canvas = document.createElement('canvas');
    const ctx = canvas.getContext('2d');
    const img = new Image();
    
    img.onload = () => {
      // 最大宽度 1920px
      const scale = Math.min(1, 1920 / img.width);
      canvas.width = img.width * scale;
      canvas.height = img.height * scale;
      ctx.drawImage(img, 0, 0, canvas.width, canvas.height);
      
      canvas.toBlob(resolve, 'image/jpeg', quality);
    };
    
    img.src = URL.createObjectURL(file);
  });
};
```

#### 资源加载策略
```html
<!-- 关键 CSS 内联 -->
<style>/* 首屏关键样式 */</style>

<!-- 非关键 CSS 异步加载 -->
<link rel="preload" href="styles.css" as="style" onload="this.rel='stylesheet'">

<!-- 图片懒加载 -->
<img data-src="image.jpg" class="lazyload" alt="...">

<!-- 脚本延迟加载 -->
<script src="app.js" defer></script>
```

### 无障碍支持

#### 基础要求

| 要求 | 说明 | 实现方式 |
|------|------|---------|
| **对比度** | 文字对比度≥4.5:1 | 使用对比度检测工具 |
| **焦点可见** | 键盘焦点明确 | :focus 样式 |
| **语义化** | 正确使用 HTML 标签 | button/a/input 等 |
| **替代文本** | 图片有 alt 描述 | alt 属性 |
| **表单标签** | 输入框有 label | label/aria-label |

#### 无障碍实现
```html
<!-- 语义化按钮 -->
<button type="button" aria-label="提交订单">
  提交订单
</button>

<!-- 图片替代文本 -->
<img src="漏水示意图.jpg" alt="卫生间漏水位置示意图">

<!-- 表单标签 -->
<label for="phone">手机号</label>
<input type="tel" id="phone" aria-required="true">

<!-- 加载状态 -->
<div role="status" aria-live="polite">
  正在分析中...
</div>
```

## 适配总结

### 核心原则

1. **移动优先** - 先设计手机版，再适配大屏
2. **渐进增强** - 基础功能全平台可用，高级功能按需支持
3. **性能优先** - 加载快、响应快、体验流畅
4. **用户友好** - 单手操作、大按钮、清晰反馈
5. **兼容并包** - 支持主流浏览器和系统版本

### 关键指标

| 指标 | 目标值 | 验收方式 |
|------|--------|---------|
| 浏览器覆盖率 | ≥95% | 用户访问统计 |
| 首屏加载时间 | <3 秒 | 性能测试 |
| 交互响应时间 | <100ms | 性能测试 |
| 无障碍合规 | WCAG AA | 无障碍检测 |
| 深色模式支持 | 完整支持 | 视觉验收 |
