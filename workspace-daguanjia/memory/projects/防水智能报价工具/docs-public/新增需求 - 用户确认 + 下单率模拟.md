## 🔴 新增需求（2 项）

### 需求 1：第一阶段增加用户确认环节

**目的**: 在用户提交信息后、留资前，增加确认环节，提升转化质量和用户信任度

**流程调整**:

**原流程**:
```
用户提交 → AI 分析 → 施工方案 → 报价 → 留资 → 完成
```

**新流程**:
```
用户提交 → AI 分析 → 施工方案 → 报价 → 🔴用户确认 → 留资 → 完成
```

**确认内容**（待小马设计）:
- 漏点位置确认
- 面积/病害确认
- 施工方案确认
- 价格确认
- 期望施工时间确认

**用户操作**:
- 确认 → 进入留资表单
- 修改 → 返回对应页面修改
- 取消 → 退出流程
