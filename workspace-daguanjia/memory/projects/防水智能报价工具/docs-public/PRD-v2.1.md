## 📊 战略背景

### 市场机遇
- **市场规模**: 200-300 亿家庭防水市场
- **线上渗透**: 约 15%（30-45 亿）
- **用户需求**: 透明报价、快速响应、专业服务

### 行业痛点
| 痛点 | 用户视角 | 平台视角 |
|------|---------|---------|
| 报价不透明 | 不知道合理价格范围 | 获客成本高，转化率低 |
| 师傅随意报价 | 担心被坑，信任度低 | 服务质量难以标准化 |
| 决策周期长 | 多家对比，犹豫不决 | 线索流失率高 |

### 战略价值
- **获客突破**: 降低获客成本，提升转化率
- **品牌信任**: 透明报价建立用户信任
- **数据积累**: 积累报价数据，优化算法

## 📱 产品方案

### 6 步用户流程

```
P1 选择漏点位置 → P2 上传现场图片 → P3 AI 智能分析 → P4 推荐施工方案 → P5 显示价格区间 → P6 确认下单
```

### 功能需求清单

#### P0 功能（必须）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| **漏点位置选择** | 卫生间/厨房/阳台/屋顶/外墙/飘窗/其他 | P0 |
| **现场图片上传** | 1-5 张照片，支持拍照/相册 | P0 |
| **AI 图像分析** | 识别漏水严重程度（轻微/普通/严重） | P0 |
| **智能方案推荐** | 根据严重程度匹配施工方案 | P0 |
| **价格区间显示** | 显示价格范围 + 构成说明 | P0 |
| **确认下单** | 填写联系信息，提交工单 | P0 |
| **工单推送** | 推送到现有工单系统 | P0 |

#### P1 功能（重要）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| **进度提示** | AI 分析时显示实时进度 | P1 |
| **价格保障说明** | 透明报价、官方质保说明 | P1 |
| **用户协议** | 协议勾选 + 隐私政策 | P1 |
| **上门时间选择** | 用户可选择期望上门时间 | P1 |

#### P2 功能（优化）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| **历史案例展示** | 展示类似案例和价格 | P2 |
| **师傅评价展示** | 展示师傅评分和评价 | P2 |
| **分享功能** | 分享给家人朋友 | P2 |

## 📊 报价算法

### 算法逻辑

```
最终价格 = 基础价格 × 位置系数 × 难度系数 × 渗漏系数 + 附加费用
```

### 系数表（待业务专家确认）

#### 位置系数

| 位置 | 系数 | 说明 |
|------|------|------|
| 卫生间 | 1.0 | 标准施工 |
| 厨房 | 1.0 | 标准施工 |
| 阳台 | 0.9 | 施工难度较低 |
| 屋顶 | 1.2 | 高空作业 |
| 外墙 | 1.3 | 高空作业 + 难度高 |
| 飘窗 | 1.1 | 难度中等 |
| 其他 | 1.0 | 默认 |
