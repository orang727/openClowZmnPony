### P0-6: 面积因子缺失 ✅

**问题描述**: 公式中无面积变量，基础价格按固定面积

**修订内容**:
- 增加面积因子计算逻辑
- 基础价格对应标准面积，超出部分按比例计算

**面积因子公式**:
```
面积因子 = 1 + (实际面积 - 标准面积) / 标准面积 × 0.5
```

**示例**:
- 卫生间 8㎡（标准 6㎡）：面积因子 = 1 + (8-6)/6 × 0.5 = 1.17
- 屋顶 30㎡（标准 20㎡）：面积因子 = 1 + (30-20)/20 × 0.5 = 1.25

**影响章节**: 报价算法

**修订位置**: 
- `📊 报价算法` → `面积因子（v2.3 新增）`

**状态**: ⏳ 待业务专家确认（截止 3/10）
