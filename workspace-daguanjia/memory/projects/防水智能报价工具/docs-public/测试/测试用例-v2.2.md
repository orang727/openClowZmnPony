## 📋 测试范围

### 7 步用户流程

```
P1 选择漏点位置 → P2 上传现场图片 → P3 面积估算 → P4 AI 智能分析 → 
P5 推荐施工方案 → P6 显示价格区间 → P7 留资并确认下单
```

### 测试重点（UI 评审优先）

| 优先级 | 测试类型 | 覆盖内容 |
|--------|---------|---------|
| **P0** | UI 交互验收 | 7 步流程界面完整、交互流畅 |
| **P0** | 功能验收 | 核心功能可用、数据流转正确 |
| **P1** | 异常流程 | 关键异常场景处理 |
| **P2** | 性能验收 | 页面加载、响应时间 |
