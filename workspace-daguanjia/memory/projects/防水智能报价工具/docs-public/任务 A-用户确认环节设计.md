## 一、新流程设计

### 当前流程 vs 新流程

**原流程**:
```
用户提交 → AI 分析 → 施工方案 → 报价 → 留资 → 完成
```

**新流程**（增加用户确认）:
```
用户提交 → AI 分析 → 施工方案 → 报价 → 🔴用户确认 → 留资 → 完成
```

**变更价值**:
1. 提升用户信任感（信息透明）
2. 减少后期纠纷（事前确认）
3. 提升留资转化率（沉没成本效应）
4. 收集更准确的用户需求

## 五、确认后流向

### 推荐方案：确认 → 留资表单

```
用户确认完成
   ↓
留资表单（简化版）
   ├─ 姓名 *
   ├─ 手机号 *
   ├─ 详细地址 *
   └─ 备注（选填）
   ↓
提交成功 → 生成工单 → 客服联系
```

**理由**:
1. 用户已完成 4 步确认，沉没成本高，留资意愿强
2. 确认信息已收集，留资表单可简化
3. 避免过早留资导致流失

### 备选方案：确认 → 直接下单（需支付定金）

```
用户确认完成
   ↓
支付定金（如 99 元）
   ↓
生成正式订单 → 师傅上门
```

**适用场景**: 紧急需求型用户（正在漏水）
