### 决策 1：用户流程

**选项 A（小马版）**: 问题类型 → 面积 → 程度 → 报价 → 留资 → 客服跟进

**选项 B（当前版）**: 位置 → 上传图片 → AI 分析 → 方案推荐 → 价格 → 确认下单

**建议**: 保持当前版（AI 分析是核心差异化）

**决策**: ⏳ 待泊宁确认

### 决策 2：建筑信息字段

**选项 A**: 添加楼层/房龄（可选字段）

**选项 B**: 保持简化（仅位置 + 图片）

**建议**: 保持简化（减少用户负担，提高转化率）

**决策**: ⏳ 待泊宁确认

---

### 决策 3：技术栈

**建议**: 采纳小马版（技术栈明确，可执行性强）
