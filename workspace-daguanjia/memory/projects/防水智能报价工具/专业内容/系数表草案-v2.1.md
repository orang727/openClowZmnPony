## 📐 系数组合规则

### 默认公式

```
最终价格 = (基础材料费 + 人工费 + 措施费) × 位置系数 × 难度系数 × 材料等级系数 × 工艺系数 + 附加费用
```

### 系数叠加规则

| 规则 | 说明 | 示例 |
|------|------|------|
| **连乘** | 所有系数相乘 | 1.0 × 1.2 × 1.0 × 1.0 = 1.2 |
| **上限** | 总系数不超过 3.0 | 避免价格过高 |
| **下限** | 总系数不低于 0.6 | 避免价格过低 |

### 示例计算

**场景**: 卫生间管根渗漏（普通）+ 标准型材料 + 局部开挖

```
基础价格：1000 元
位置系数：1.0（卫生间）
难度系数：1.2（管根渗漏 - 普通）
材料系数：1.0（标准型）
工艺系数：1.0（局部开挖）

最终价格 = 1000 × 1.0 × 1.2 × 1.0 × 1.0 = 1200 元
```

**场景**: 外墙表面渗水（严重）+ 高端型材料 + 特殊工艺

```
基础价格：3000 元
位置系数：1.3（外墙）
难度系数：1.8（表面渗水 - 严重）
材料系数：1.5（高端型）
工艺系数：2.0（特殊工艺）

总系数 = 1.3 × 1.8 × 1.5 × 2.0 = 7.02 → 超过上限 3.0，按 3.0 计算
最终价格 = 3000 × 3.0 = 9000 元
```

### 讨论点（3/10 Workshop）

- [ ] **系数上限 3.0 是否合理？**（极端场景可能不够）
- [ ] **是否采用"取最大值"而非"连乘"？**（避免系数叠加过高）
- [ ] **附加费用是否应纳入系数计算？**（紧急施工、远程费等）
