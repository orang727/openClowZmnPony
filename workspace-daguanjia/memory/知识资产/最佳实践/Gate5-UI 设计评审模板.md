### P1. 漏点位置选择页面

**页面元素**:
- [ ] 页面标题："请选择漏点位置"
- [ ] 位置选项卡片（7 个）：
  - 卫生间
  - 厨房
  - 阳台
  - 屋顶

## 3️⃣ 交互原型

### 用户流程

```
P1 选择位置 → P2 上传图片 → P3 AI 分析 → P4 方案推荐 → P5 价格显示 → P6 确认下单
     ↓              ↓            ↓           ↓           ↓           ↓
   可返回        可返回       不可返回     可返回      可返回      可返回
```

### 交互说明

| 交互 | 说明 | 动效 |
|------|------|------|
| 页面切换 | 从右向左滑动 | 300ms ease |
| 按钮点击 | 按下有反馈 | 100ms scale |
| 卡片选择 | 选中高亮 | 200ms color |
| 图片上传 | 上传进度显示 | 进度条动画 |
| AI 分析 | 加载动画 | 循环动画 |

### UI 设计评审评分表（满分 10 分）

| 维度 | 权重 | 评分 (0-10) | 得分 |
|------|------|-----------|------|
| **美观性** | 25% | - | - |
| **一致性** | 25% | - | - |
| **可用性** | 25% | - | - |
| **品牌匹配** | 15% | - | - |
| **可实现性** | 10% | - | - |
| **总分** | 100% | - | **-/10** |

### 评审标准

| 分数 | 结论 | 说明 |
|------|------|------|
| ≥8 分 | ✅ 通过 | 可交付开发 |
