## 1. 核心说明

### 1.1 价格性质声明

> ⚠️ **重要提示**: P5 页面显示的价格为 **"估算价格范围"**，非最终成交价格。

- 该价格基于用户提交的病害信息、位置、面积等数据进行算法估算
- 实际施工条件可能存在差异，需工程师上门评估后确定最终价格
- 估算价格仅供参考，不作为合同定价依据

### 1.2 免责声明

> **最终价格以上门评估为准**

## 2. 价格计算公式

### 2.1 基础公式

```
估算价格 = (材料费 + 人工费 + 措施费) × 位置系数 × 难度系数 + 附加费用
```

### 2.2 公式拆解

| 组成部分 | 说明 | 计算方式 |
|---------|------|---------|
| **材料费** | 防水材料、辅材成本 | 单价 × 用量 × (1 + 损耗率) |
| **人工费** | 施工人员工时成本 | 工时 × 工时单价 |
| **措施费** | 保护措施、清理费用 | 固定费用或按面积计算 |
| **位置系数** | 施工位置难度调整 | 查系数表（见第 3 节） |
| **难度系数** | 病害复杂程度调整 | 查系数表（见第 3 节） |
| **附加费用** | 特殊工艺、加急费等 | 按实际发生项累加 |

### 2.3 价格范围计算

```
最低估算价 = 基础价格 × 0.9 × 位置系数 × 难度系数 + 附加费用
最高估算价 = 基础价格 × 1.1 × 位置系数 × 难度系数 + 附加费用
```

**显示格式**: `¥{最低估算价} - ¥{最高估算价}`

## 3. 系数表引用

### 3.1 系数表版本

> 📋 **引用文档**: 《系数表草案 v2.1》

### 3.2 位置系数表（摘要）

| 施工位置 | 系数 | 说明 |
|---------|------|------|
| 地面/平面 | 1.0 | 标准施工条件 |
| 墙面/立面 | 1.2 | 需高空作业保护 |
| 屋顶 | 1.3 | 高空作业 + 材料运输 |
| 地下室 | 1.4 | 空间受限 + 通风要求 |
| 卫生间 | 1.1 | 空间狭小 + 成品保护 |
| 厨房 | 1.1 | 空间狭小 + 设备保护 |
| 阳台 | 1.2 | 高空作业风险 |
| 电梯井 | 1.5 | 特殊作业环境 |

### 3.3 难度系数表（摘要）

| 病害严重程度 | 系数 | 说明 |
|-------------|------|------|
| 轻度（表面渗水） | 1.0 | 标准处理流程 |
| 中度（明显渗漏） | 1.2 | 需局部开挖/注浆 |
| 重度（结构渗漏） | 1.4 | 复杂工艺 + 多道工序 |
| 极重（结构性破坏） | 1.6 | 需结构加固 + 防水 |

### 3.4 系数表获取

- 完整系数表存储于：`docs/technical/系数表草案-v2.1.md`
- 系数表更新需经技术委员会审批
- 前端通过 API `/api/coefficients` 获取最新系数

## 4. 附加费用项

### 4.1 常见附加费用

| 费用项 | 触发条件 | 计费方式 |
|-------|---------|---------|
| 高空作业费 | 施工高度>2 米 | ¥200/人/天 |
| 夜间施工费 | 18:00-8:00 施工 | 人工费×1.5 |
| 加急服务费 | 48 小时内上门 | 总价×1.3 |
| 成品保护费 | 需保护家具/设备 | ¥100-500/间 |
| 垃圾清运费 | 产生建筑垃圾 | ¥200/车 |
| 特殊工艺费 | 注浆/灌浆等 | 按实际发生 |

### 4.2 附加费用计算

```
附加费用 = Σ(各触发项费用)
```

## 5. "预约顾问获取精确报价"触发条件

### 5.1 触发规则

当满足以下**任一条件**时，显示"预约顾问获取精确报价"按钮：

| 条件 | 判断逻辑 | 提示文案 |
|-----|---------|---------|
| 面积过大 | 估算面积 > 500㎡ | "面积较大，建议预约顾问上门评估" |
| 病害复杂 | 病害类型 ≥ 3 种 | "多种病害并存，需专业诊断" |
| 难度过高 | 难度系数 > 1.4 | "施工难度较高，需现场勘测" |
| 特殊位置 | 位置 = 电梯井/泳池/隧道 | "特殊施工环境，需定制方案" |
| 价格超限 | 估算总价 > ¥50,000 | "大额工程，建议面谈详细方案" |
| 用户主动请求 | 用户点击"预约顾问" | - |

### 5.2 前端交互逻辑

```javascript
function shouldShowConsultantButton(estimate) {
  const conditions = [
    estimate.area > 500,
    estimate.diseaseTypes >= 3,
    estimate.difficultyCoefficient > 1.4,
    ['电梯井', '泳池', '隧道'].includes(estimate.location),
    estimate.totalPrice > 50000
  ];
  
  return conditions.some(c => c === true);
}
```

### 5.3 按钮文案

- **默认**: "预约顾问获取精确报价"
- **高难度**: "预约专家上门评估（免费）"
- **大额**: "预约高级顾问定制方案"

## 6. 价格计算流程

### 6.1 计算时序

```
用户提交信息 
  ↓
前端校验完整性
  ↓
调用 /api/price/estimate
  ↓
后端查询系数表
  ↓
计算基础价格
  ↓
应用系数调整
  ↓
累加附加费用
  ↓
返回价格范围
  ↓
前端展示 + 触发条件判断
```

### 6.2 API 响应示例

```json
{
  "code": 200,
  "data": {
    "priceRange": {
      "min": 8500,
      "max": 10500,
      "display": "¥8,500 - ¥10,500"

      "measureCost": 500,
      "locationCoefficient": 1.2,
      "difficultyCoefficient": 1.1,
      "additionalCosts": 500
    },
    "disclaimer": "最终价格以上门评估为准",
    "showConsultantButton": false
  }
}
```
