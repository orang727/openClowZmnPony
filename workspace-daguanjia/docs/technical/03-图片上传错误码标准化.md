# 图片上传错误码标准化文档

**版本**: v1.0  
**创建时间**: 2026-03-08  
**适用范围**: 全平台图片上传功能

---

## 1. 错误码分类总览

### 1.1 错误码编码规则

```
错误码格式：XXXX
- 第 1 位：错误大类 (1-5)
- 后 3 位：具体错误序号 (001-099)
```

### 1.2 五大错误类别

| 大类 | 编码范围 | 错误类型 | 说明 |
|-----|---------|---------|------|
| 1 | 1001-1099 | 网络错误 | 上传过程中的网络连接问题 |
| 2 | 2001-2099 | 格式错误 | 图片格式不支持或损坏 |
| 3 | 3001-3099 | 大小错误 | 图片尺寸/文件大小超限 |
| 4 | 4001-4099 | 数量错误 | 上传数量超出限制 |
| 5 | 5001-5099 | 其他错误 | 服务端异常、权限问题等 |

## 2. 网络错误（1001-1099）

### 2.1 错误码明细

| 错误码 | 错误名称 | 触发条件 | 用户提示文案 |
|-------|---------|---------|-------------|
| 1001 | NETWORK_TIMEOUT | 上传请求超时 (>30s) | "网络连接超时，请检查网络后重试" |
| 1002 | NETWORK_DISCONNECTED | 网络断开 | "网络已断开，请检查网络连接" |
| 1003 | NETWORK_UNSTABLE | 网络不稳定 | "网络不稳定，建议切换 WiFi 后重试" |
| 1004 | SERVER_UNREACHABLE | 服务器无法连接 | "服务器暂时无法连接，请稍后重试" |
| 1005 | UPLOAD_INTERRUPTED | 上传中断 | "上传中断，请重新上传" |
| 1006 | DNS_RESOLUTION_FAILED | DNS 解析失败 | "无法解析服务器地址，请检查网络设置" |
| 1007 | SSL_HANDSHAKE_FAILED | SSL 握手失败 | "安全连接失败，请检查系统时间设置" |
| 1008 | PROXY_ERROR | 代理错误 | "代理配置错误，请检查网络设置" |
| 1009 | BANDWIDTH_EXCEEDED | 带宽超限 | "当前网络繁忙，请稍后重试" |
| 1010 | CONNECTION_RESET | 连接被重置 | "连接被重置，请重新尝试上传" |

### 2.2 重试机制

| 错误码 | 是否自动重试 | 重试次数 | 重试间隔 | 用户操作建议 |
|-------|------------|---------|---------|-------------|
| 1001 | ✅ 是 | 3 次 | 2s, 4s, 8s (指数退避) | 等待自动重试，或手动点击重试 |
| 1002 | ❌ 否 | - | - | 检查网络连接后手动重试 |
| 1003 | ✅ 是 | 2 次 | 3s, 6s | 建议切换网络后重试 |
| 1004 | ✅ 是 | 3 次 | 5s, 10s, 20s | 等待后自动重试 |
| 1005 | ✅ 是 | 2 次 | 2s, 4s | 检查网络后手动重试 |
| 1006-1010 | ❌ 否 | - | - | 根据提示检查设置后重试 |

### 2.3 前端处理逻辑

```javascript
// 网络错误重试配置
const NETWORK_RETRY_CONFIG = {
  1001: { autoRetry: true, maxRetries: 3, delays: [2000, 4000, 8000] },
  1002: { autoRetry: false, maxRetries: 0, delays: [] },
  1003: { autoRetry: true, maxRetries: 2, delays: [3000, 6000] },
  1004: { autoRetry: true, maxRetries: 3, delays: [5000, 10000, 20000] },
  1005: { autoRetry: true, maxRetries: 2, delays: [2000, 4000] },
};

async function uploadWithRetry(file, errorCode = null) {
  const config = NETWORK_RETRY_CONFIG[errorCode] || { autoRetry: false };
  
  if (!config.autoRetry) {
    return await uploadFile(file);
  }
  
  for (let i = 0; i < config.maxRetries; i++) {
    try {
      return await uploadFile(file);
    } catch (error) {
      if (i === config.maxRetries - 1) throw error;
      
      // 指数退避等待
      await sleep(config.delays[i]);
    }
  }
}
```

## 3. 格式错误（2001-2099）

### 3.1 错误码明细

| 错误码 | 错误名称 | 触发条件 | 用户提示文案 |
|-------|---------|---------|-------------|
| 2001 | UNSUPPORTED_FORMAT | 不支持的图片格式 | "不支持该图片格式，请上传 JPG/PNG/GIF 格式" |
| 2002 | CORRUPTED_FILE | 文件损坏 | "图片文件已损坏，请重新选择图片" |
| 2003 | INVALID_IMAGE | 无效图片 | "这不是有效的图片文件，请重新选择" |
| 2004 | MISSING_EXTENSION | 缺少文件扩展名 | "文件名缺少扩展名，请添加.jpg 或.png 后缀" |
| 2005 | MIME_TYPE_MISMATCH | MIME 类型不匹配 | "文件类型与扩展名不符，请检查后重新上传" |
| 2006 | ENCODING_ERROR | 编码错误 | "图片编码错误，请重新保存后上传" |
| 2007 | COLOR_SPACE_UNSUPPORTED | 不支持的色彩空间 | "不支持该色彩模式，请使用 RGB 模式" |
| 2008 | BIT_DEPTH_UNSUPPORTED | 不支持的位深度 | "不支持该位深度，请使用 8 位或 24 位图片" |
| 2009 | METADATA_CORRUPTED | 元数据损坏 | "图片信息异常，请重新保存后上传" |
| 2010 | PROGRESSIVE_JPEG | 渐进式 JPEG 不支持 | "请使用标准 JPEG 格式，重新保存图片" |

### 3.2 支持的图片格式

| 格式 | 扩展名 | MIME 类型 | 最大支持 |
|-----|-------|----------|---------|
| JPEG | .jpg, .jpeg | image/jpeg | ✅ 支持 |
| PNG | .png | image/png | ✅ 支持 |
| GIF | .gif | image/gif | ✅ 支持 (静态) |
| WebP | .webp | image/webp | ✅ 支持 |
| BMP | .bmp | image/bmp | ❌ 不支持 |
| TIFF | .tif, .tiff | image/tiff | ❌ 不支持 |
| HEIC | .heic | image/heic | ❌ 不支持 (iOS 需转换) |
| RAW | .raw, .cr2, .nef | - | ❌ 不支持 |

### 3.3 重试机制

| 错误码 | 是否自动重试 | 重试次数 | 用户操作建议 |
|-------|------------|---------|-------------|
| 2001-2003 | ❌ 否 | - | 更换图片后重新上传 |
| 2004 | ❌ 否 | - | 修改文件名后重新上传 |
| 2005-2010 | ❌ 否 | - | 重新保存图片后上传 |

**说明**: 格式错误通常无法通过重试解决，需要用户更换或重新处理图片。

### 3.4 前端预处理建议

```javascript
// 上传前格式校验
function validateImageFormat(file) {
  const allowedTypes = ['image/jpeg', 'image/png', 'image/gif', 'image/webp'];
  const allowedExtensions = ['.jpg', '.jpeg', '.png', '.gif', '.webp'];
  
  // 检查 MIME 类型
  if (!allowedTypes.includes(file.type)) {
    return { valid: false, errorCode: 2001 };
  }
  
  // 检查扩展名
  const ext = '.' + file.name.split('.').pop().toLowerCase();
  if (!allowedExtensions.includes(ext)) {
    return { valid: false, errorCode: 2004 };
  }
  
  return { valid: true };
}

// iOS HEIC 转换提示
function checkHEIC(file) {
  if (file.type === 'image/heic' || file.name.toLowerCase().endsWith('.heic')) {
    return {
      valid: false,
      errorCode: 2001,
      message: '检测到 HEIC 格式，请在 iPhone 设置中选择"兼容性最佳"后重新拍摄，或使用转换工具转为 JPG 格式'
    };
  }
}
```

### 4.4 前端自动压缩逻辑

```javascript
// 图片自动压缩
async function compressImage(file, maxSize = 10 * 1024 * 1024) {
  if (file.size <= maxSize) {
    return file;
  }
  
  return new Promise((resolve) => {
    const img = new Image();
    img.src = URL.createObjectURL(file);
    img.onload = () => {
      const canvas = document.createElement('canvas');
      const ctx = canvas.getContext('2d');
      
      // 计算压缩后的尺寸
      let width = img.width;
      let height = img.height;
      const maxDimension = 2048;
      
      if (width > maxDimension || height > maxDimension) {
        const ratio = maxDimension / Math.max(width, height);
        width = Math.floor(width * ratio);
        height = Math.floor(height * ratio);
      }
      
      canvas.width = width;
      canvas.height = height;
      ctx.drawImage(img, 0, 0, width, height);
      
      // 压缩质量调整
      let quality = 0.9;
      canvas.toBlob((blob) => {
        if (blob.size > maxSize && quality > 0.5) {
          quality -= 0.1;
          // 递归压缩
          canvas.toBlob(resolve, 'image/jpeg', quality);
        } else {
          resolve(new File([blob], file.name, { type: 'image/jpeg' }));
        }
      }, 'image/jpeg', quality);
    };
  });
}
```

### 5.4 前端数量校验逻辑

```javascript
// 上传前数量校验
function validateFileCount(files, maxCount = 9) {
  if (files.length === 0) {
    return { valid: false, errorCode: 4002 };
  }
  
  if (files.length > maxCount) {
    return { 
      valid: false, 
      errorCode: 4001,
      message: `最多可上传${maxCount}张图片，当前已选择${files.length}张`
    };
  }
  
  // 检查重复文件
  const uniqueFiles = new Set(files.map(f => `${f.name}-${f.size}-${f.lastModified}`));
  if (uniqueFiles.size !== files.length) {
    return { valid: false, errorCode: 4003 };
  }
  
  return { valid: true };
}

// 并发控制
class UploadQueue {
  constructor(concurrency = 3) {
    this.concurrency = concurrency;
    this.running = 0;
    this.queue = [];
  }
  
  async add(task) {
    if (this.running < this.concurrency) {
      this.running++;
      try {
        return await task();
      } finally {
        this.running--;
        this.next();
      }
    } else {
      return new Promise(resolve => this.queue.push(resolve));
    }
  }
  
  next() {
    if (this.queue.length > 0 && this.running < this.concurrency) {
      const resolve = this.queue.shift();
      resolve();
    }
  }
}
```

```javascript
// 错误上报
async function reportError(errorCode, context) {
  const errorReport = {
    errorCode,
    timestamp: Date.now(),
    userAgent: navigator.userAgent,
    networkType: await getNetworkType(),
    ...context
  };
  
  // 5xxx 错误需要上报服务端分析

    await fetch('/api/error/report', {
      method: 'POST',
      body: JSON.stringify(errorReport)
    });
  }
  
  // 记录到本地日志
  console.error(`[Upload Error] ${errorCode}`, errorReport);
}
```

## 7. 错误处理流程图

```
用户上传图片
    ↓
前端预校验 (格式/大小/数量)
    ↓
校验通过？──否──→ 显示对应错误 (2xxx/3xxx/4xxx)
    ↓是
发起上传请求
    ↓
上传成功？──否──→ 判断错误类型
                        ↓
            ┌───────────┼───────────┐
            ↓           ↓           ↓
        网络错误     服务端错误     其他错误
        (1xxx)      (5xxx)       (2xxx/3xxx/4xxx)
            ↓           ↓           ↓
        自动重试？自动重试？用户处理
            ↓           ↓
        重试成功？重试成功？
            ↓           ↓
    ┌───────┴───┐   ┌───┴───────┐
    ↓           ↓   ↓           ↓
  成功       失败  成功       失败
    ↓           ↓   ↓           ↓
  完成       显示  完成       显示
  上传       错误  上传       错误
```
