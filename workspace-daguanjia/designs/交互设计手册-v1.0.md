## 3. 组件库

```css
.btn-primary {
  width: 100%;
  height: 52px;

  color: #FFFFFF;
  border: none;
  border-radius: 14px;
```

## 4. 交互规范

### 4.1 页面转场

| 类型 | 动画 | 时长 | 使用场景 |
|------|------|------|----------|
| Push | 从右滑入 | 0.3s | 进入新页面 |
| Pop | 从左滑出 | 0.3s | 返回上一页 |
| Fade | 淡入淡出 | 0.2s | 弹窗、对话框 |

### 4.2 按钮反馈

```css
.btn:active {
  transform: scale(0.98);
}
```

### 4.3 表单验证

- **实时验证**: 输入时即时反馈
- **错误提示**: 红色边框 + 错误文字
- **成功提示**: 绿色边框 + 成功图标

### 4.4 加载状态

- **下拉刷新**: 旋转图标 + 文字提示
- **骨架屏**: 闪烁动画，1.5s 循环
- **Loading**: 旋转 Spinner，1s 循环

## 5. 使用示例

### 5.1 完整页面示例

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>示例页面</title>
    <style>
        :root {
            --primary: #1E40AF;
            --primary-light: #3B82F6;
            --bg-page: #F8F9FC;
            --bg-card: #FFFFFF;
        }
        
        body {
            font-family: -apple-system, BlinkMacSystemFont, 'PingFang SC', sans-serif;
            background: var(--bg-page);
            margin: 0;
            padding: 0;
        }
        
        .header {
            background: linear-gradient(135deg, var(--primary) 0%, var(--primary-light) 100%);
            color: white;
            padding: 55px 20px 20px;
            text-align: center;
        }
        
        .header h1 {
            font-size: 20px;
            font-weight: 600;
            margin: 0;
        }
        
        .content {
            padding: 20px 16px;
        }
        
        .card {
            background: var(--bg-card);
            border-radius: 16px;
            padding: 20px;
            margin-bottom: 16px;
            box-shadow: 0 2px 12px rgba(0,0,0,0.06);
        }
        
        .btn {
            width: 100%;
            height: 52px;
            background: linear-gradient(135deg, var(--primary) 0%, var(--primary-light) 100%);
            color: white;
            border: none;
            border-radius: 14px;
            font-size: 16px;
            font-weight: 600;
            cursor: pointer;
        }
        
        .input {
            width: 100%;
            height: 52px;
            padding: 0 16px;
            border: 1px solid #E5E7EB;
            border-radius: 12px;
            font-size: 15px;
            margin-bottom: 16px;
        }
    </style>
</head>
<body>
    <div class="header">
        <h1>页面标题</h1>
    </div>
    <div class="content">
        <div class="card">
            <h3 style="margin-top:0">卡片标题</h3>
            <input class="input" placeholder="请输入内容">
            <button class="btn">主要按钮</button>
        </div>
    </div>
</body>
</html>
```

### 5.2 CSS 变量使用

```css
:root {
  /* 主色 */
  --color-primary: #1E40AF;
  --color-primary-light: #3B82F6;
  
  /* 功能色 */
  --color-success: #10B981;
  --color-warning: #F59E0B;
  --color-error: #EF4444;
  
  /* 中性色 */
  --color-text-primary: #1F2937;
  --color-text-secondary: #6B7280;
  --color-border: #E5E7EB;
  --color-bg-page: #F8F9FC;
  --color-bg-card: #FFFFFF;
  
  /* 间距 */
  --spacing-xs: 4px;
  --spacing-sm: 8px;
  --spacing-md: 12px;
  --spacing-lg: 16px;
  --spacing-xl: 20px;
  
  /* 圆角 */
  --radius-sm: 4px;
  --radius-md: 8px;
  --radius-lg: 12px;
  --radius-xl: 14px;
  --radius-2xl: 16px;
  --radius-3xl: 20px;
}
```
