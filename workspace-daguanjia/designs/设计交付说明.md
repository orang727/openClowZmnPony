## 🔧 Pencil MCP 调用示例

必须通过 mcporter 调用，禁止直接调用二进制文件：

```bash
# 1. 创建新文档
mcporter call pencil.open_document filePathOrNew:new

# 2. 批量创建设计元素
mcporter call pencil.batch_design --args '{"filePath": "new", "operations": "screen=I(document,{type:\"frame\",width:375,height:812,fill:\"#f8f9fc\"})\nheader=I(screen,{type:\"frame\",width:375,height:140,fill:\"#1e40af\"})"}'

# 3. 导出截图
mcporter call pencil.get_screenshot --args '{"filePath": "new", "nodeId": "screen"}'
```
