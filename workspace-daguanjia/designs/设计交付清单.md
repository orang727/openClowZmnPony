## 📱 页面设计说明

### S1 漏点位置选择
- 8 个位置卡片：卫生间/厨房/阳台/屋顶/外墙/飘窗/地下室/其他
- 每个卡片含图标、名称、示例说明
- 选中状态：蓝色边框 + 浅蓝渐变背景
- 6 步进度条，当前步骤高亮

### S2 上传现场图片
- 拍摄建议提示框（黄色警告样式）
- 照片网格预览（4 列）
- 已上传照片支持删除
- 上传区域：虚线边框 + 相机图标

### S3 AI 智能分析
- AI 置信度警告提示（<70% 显示）
- AI 分析结果卡片：面积/病害/严重程度/置信度
- 手动修正区域：长宽输入、严重程度选择
- 三档严重程度：轻微/中等/严重

### S4 施工方案推荐
- 三档方案：经济型/标准型（推荐）/尊享型
- 每档含：工艺步骤标签、材料、时间、质保期
- 选中方案：蓝色边框 + 浅蓝背景

### S5 价格显示
- 价格区间：48px 超大字体，蓝色渐变背景
- 价格构成明细：材料费/人工费/面积因子/难度系数
- AI 估价说明：黄色警告框

### S6 留资表单
- 表单项：姓名/手机/地址/期望时间/补充说明
- 必填项标注（红色*）
- 协议勾选框
- 粘性底部操作区

### 成功页
- 绿色成功图标
- 工单号展示
- 客服 15 分钟联系承诺
- 订单摘要

### 异常状态页
- 网络失败：📡 图标 + 重试按钮
- AI 识别失败：🤖 图标 + 拍摄建议 + 手动输入入口
- 上传失败：📸 图标 + 失败文件提示 + 重试/跳过

## 🔧 Pencil MCP 使用说明

### 调用方式
```bash
# 1. 创建新文档
echo '{"jsonrpc":"2.0","id":1,"method":"tools/call","params":{"name":"open_document","arguments":{"filePathOrTemplate":"new"}}}' | /Applications/Pencil.app/Contents/Resources/app.asar.unpacked/out/mcp-server-darwin-arm64 --app desktop

# 2. 批量创建设计元素
echo '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"batch_design","arguments":{"filePath":"new","operations":"screen=I(document, {type: \"frame\", x: 0, y: 0, width: 375, height: 812, fill: \"#f8f9fc\"})\nheader=I(screen, {type: \"frame\", x: 0, y: 0, width: 375, height: 140, fill: \"#1e40af\"})"}}}' | /Applications/Pencil.app/Contents/Resources/app.asar.unpacked/out/mcp-server-darwin-arm64 --app desktop

# 3. 导出截图
echo '{"jsonrpc":"2.0","id":3,"method":"tools/call","params":{"name":"get_screenshot","arguments":{"filePath":"new","nodeId":"screen"}}}' | /Applications/Pencil.app/Contents/Resources/app.asar.unpacked/out/mcp-server-darwin-arm64 --app desktop
```

### 注意事项
- Pencil MCP 需要应用内 WebSocket 授权
- `batch_design` 操作建议使用 flexbox 布局，避免绝对定位
- 截图导出为 base64 PNG，需解码保存
