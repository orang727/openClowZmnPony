| 页面 | 名称 | 核心功能 |

## 🎨 设计规范

### 色彩系统
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#1e40af` → `#3b82f6` | 蓝色渐变，专业信赖 |
| 成功色 | `#10b981` | 完成状态、成功提示 |
| 警告色 | `#f59e0b` | 中等严重度、AI 置信度提示 |
| 危险色 | `#ef4444` | 严重渗漏、错误状态 |
| 背景色 | `#f8f9fc` | 页面背景 |
| 卡片背景 | `#ffffff` | 内容卡片 |

### 字体规范
| 元素 | 字号 | 字重 | 说明 |
|------|------|------|------|
| 页面标题 | 22px | 600 | 顶部导航 |
| 卡片标题 | 16px | 600 | 内容分区 |
| 正文 | 14-15px | 400 | 常规文本 |
| 辅助文本 | 12-13px | 400 | 说明、提示 |
| 价格区间 | 48px | 800 | 超大突出显示 |
| 价格明细 | 14-20px | 600-700 | 分级显示 |

### 组件规范
- **卡片圆角**: 16px
- **按钮圆角**: 14px
- **输入框圆角**: 12px
- **进度条圆点**: 26px
- **手机边框**: 14px 黑色，圆角 44px

### 交互规范
- **步骤进度条**: 6 步横向排列，完成/当前/未完成三态
- **返回按钮**: 每页左上角圆形返回键
- **底部操作区**: 粘性定位，包含上一步/下一步按钮
- **选中状态**: 蓝色边框 + 浅蓝背景渐变

## 📊 设计亮点

### 1. 清晰的流程引导
- 6 步进度条贯穿全程
- 每步显示当前进度和已完成步骤
- 支持随时返回上一步

### 2. 价格突出展示
- S5 价格区间使用 48px 超大字体
- 蓝色渐变背景突出价格区域
- 价格构成明细清晰分项展示

### 3. AI 置信度处理
- 置信度<70% 时显示黄色警告提示
- 提供 AI/手动双模式切换入口
- 手动修正支持面积长宽输入

### 4. 异常状态完善
- 网络失败：重试 + 返回首页
- AI 识别失败：原因说明 + 拍摄建议 + 手动输入
- 上传失败：失败文件提示 + 重试/跳过

### 5. 专业信赖感
- 蓝色系主色调（防水行业常用）
- 简洁的卡片式布局
- 清晰的视觉层级
- 完善的提示和说明文案
