## 8. 开发注意事项

### 8.1 响应式适配

```css
/* 移动端优先 */
@media (max-width: 375px) {
  .phone {
    width: 100%;
  }
}

/* 平板适配 */
@media (min-width: 768px) {
  .pages {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* 桌面适配 */
@media (min-width: 1200px) {
  .pages {
    grid-template-columns: repeat(3, 1fr);
  }
}
```

### 8.2 安全区适配

```css
/* iOS 安全区适配 */
.safe-area {
  padding-bottom: env(safe-area-inset-bottom);
}

/* 状态栏适配 */
.status-bar {
  padding-top: env(safe-area-inset-top);
}
```

### 8.3 性能优化

1. **图片优化**: 使用 WebP 格式，压缩至 80% 质量
2. **CSS 压缩**: 生产环境使用压缩后的 CSS
3. **字体加载**: 使用系统字体，避免网络字体
4. **动画优化**: 使用 transform 和 opacity，避免重排

### 8.4 可访问性

1. **对比度**: 文本与背景对比度至少 4.5:1
2. **点击区域**: 最小点击区域 44×44px
3. **字体大小**: 正文不小于 14px
4. **颜色不依赖**: 重要信息不只用颜色区分

### 8.5 浏览器兼容性

| 浏览器 | 最低版本 | 说明 |
|--------|---------|------|
| Chrome | 80+ | 完全支持 |
| Safari | 13+ | 完全支持 |
| Firefox | 75+ | 完全支持 |
| Edge | 80+ | 完全支持 |
| iOS Safari | 13+ | 完全支持 |
| Android Chrome | 80+ | 完全支持 |
