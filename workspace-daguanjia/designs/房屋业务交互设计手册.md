## 1. 基础设计系统

### 1.1 色彩系统

基于 34 张房屋业务 UI 参考图分析，提取以下色彩规范：

#### 主色系

| 名称 | 色值 | 使用场景 |
|------|------|----------|
| 品牌蓝 | `#1E40AF` | 主按钮、品牌标识、重要操作 |
| 品牌蓝浅 | `#3B82F6` | 次级按钮、链接、高亮 |
| 渐变起始 | `#1E40AF` → `#3B82F6` | 渐变背景、主视觉区 |

#### 辅助色系

| 名称 | 色值 | 使用场景 |
|------|------|----------|
| 成功绿 | `#10B981` | 成功状态、完成标识、正面反馈 |
| 警告黄 | `#F59E0B` | 警告提示、中等风险、注意标识 |
| 危险红 | `#EF4444` | 错误状态、删除操作、高风险 |
| 信息蓝 | `#0EA5E9` | 信息提示、帮助说明 |

#### 中性色系

| 名称 | 色值 | 使用场景 |
|------|------|----------|
| 标题黑 | `#1F2937` | 主标题、重要文本 |
| 正文灰 | `#374151` | 正文内容 |
| 辅助灰 | `#6B7280` | 辅助说明、次要文本 |
| 占位灰 | `#9CA3AF` | 占位符、禁用文本 |
| 边框灰 | `#E5E7EB` | 边框、分割线 |
| 背景灰 | `#F3F4F6` | 次要背景 |
| 页面背景 | `#F8F9FC` | 页面主背景 |

#### 功能色透明变体

```css
/* 白色透明 */
--white-10: rgba(255, 255, 255, 0.10);
--white-20: rgba(255, 255, 255, 0.20);
--white-30: rgba(255, 255, 255, 0.30);

/* 黑色透明 */
--black-05: rgba(0, 0, 0, 0.05);
--black-10: rgba(0, 0, 0, 0.10);
--black-20: rgba(0, 0, 0, 0.20);
```

### 1.3 图标系统

#### 图标风格

- **风格**: 线性图标 + 面性图标结合
- **圆角**: 2px 圆角端点
- **描边**: 1.5px - 2px
- **尺寸序列**: 16px / 20px / 24px / 32px / 48px

#### 常用图标尺寸

| 场景 | 尺寸 | 使用示例 |
|------|------|----------|
| 列表图标 | 20px | 列表项前缀 |
| 功能图标 | 24px | 按钮图标、操作图标 |
| 空状态图标 | 48px - 80px | 空页面、错误页 |
| 导航图标 | 24px | 底部导航栏 |
| 状态图标 | 32px | 成功/失败/警告 |

#### 图标颜色规范

```css
/* 默认状态 */
--icon-default: #6B7280;

/* 激活状态 */
--icon-active: #1E40AF;

/* 禁用状态 */
--icon-disabled: #D1D5DB;

/* 功能色 */
--icon-success: #10B981;
--icon-warning: #F59E0B;
--icon-error: #EF4444;
```

/* 页面边距 */

| 名称 | 值 | 使用场景 |

## 3. 交互规范

### 3.1 页面转场动画

#### 推入动画 (Push)

```css
/* 新页面从右侧进入 */
.page-enter {
  transform: translateX(100%);
  opacity: 0;
}

.page-enter-active {
  transform: translateX(0);
  opacity: 1;
  transition: all 0.3s ease-out;
}

/* 旧页面向左侧退出 */
.page-exit {
  transform: translateX(0);
  opacity: 1;
}

.page-exit-active {
  transform: translateX(-20%);
  opacity: 0;
  transition: all 0.3s ease-in;
}
```

#### 淡入动画 (Fade)

```css
.page-fade-enter {
  opacity: 0;
}

.page-fade-enter-active {
  opacity: 1;
  transition: opacity 0.2s ease-in;
}
```

#### 缩放动画 (Scale)

用于弹窗、对话框：

```css
.dialog-scale-enter {
  transform: scale(0.9);
  opacity: 0;
}

.dialog-scale-enter-active {
  transform: scale(1);
  opacity: 1;
  transition: all 0.2s ease-out;
}
```
