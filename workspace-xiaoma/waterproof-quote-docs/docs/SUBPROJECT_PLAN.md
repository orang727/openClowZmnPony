**规划日期**：2026-03-05

## 一、产品定位

**核心流程**：客户自查漏水情况 → AI 识别故障 → 引导下单 → 工程师上门勘察最终确认

## 二、子项目1：图片识别（1天）

### 2.1 功能目标
用户上传现场照片，AI 自动识别病害类型和严重程度。

### 2.2 识别类型（6种）

| 病害类型 | 说明 | 典型场景 |
|----------|------|----------|
| 水渍 | 墙面/地面水渍印记 | 卫生间墙角发霉变黑 |
| 裂缝 | 墙面/地面裂缝 | 屋顶裂纹、墙体开裂 |
| 潮湿 | 局部潮湿、凝水 | 地下室墙面渗水 |
| 泛碱 | 白色碱化结晶 | 瓷砖缝、外墙起白硝 |
| 渗漏 | 明显漏水痕迹 | 水珠滴落、明显水流 |
| 发霉 | 霉菌斑块 | 黑色/绿色霉斑 |

### 2.3 严重程度判定

| 等级 | 判定依据 |
|------|----------|
| 轻微 | 局部小面积（<10cm²）、早期症状、不影响结构 |
| 普通 | 明显可见、范围适中（10-50cm²）、需要处理 |
| 严重 | 大面积（>50cm²）、已影响结构、需紧急处理 |

### 2.4 技术实现

**API 选择**：Minimax Vision API（多模态理解强，成本低）

**调用流程**：
```
用户上传图片 → 前端压缩 → 上传OSS → 调用AI识别API → 返回病害类型+严重程度 → 展示结果
```

### 2.5 每日任务

| 时间 | 任务 |
|------|------|
| 上午 | API 账号申请、接口测试 |
| 下午 | 后端封装、前端集成联调 |

## 三、子项目2：智能故障匹配（3天）

### 3.1 功能目标
根据 AI 识别结果，匹配故障类型，给出初步诊断说明。

### 3.2 漏点位置（7个）

| 位置 | 特点 |
|------|------|
| 卫生间 | 防水重点区域、管线多 |
| 厨房 | 用水频繁、管道复杂 |
| 阳台 | 雨水冲刷、日晒老化 |
| 屋顶 | 直面雨水、保温层问题 |
| 外墙 | 日晒雨淋、裂缝风险 |
| 飘窗 | 结构薄弱、渗水多发 |
| 其他 | 窗框、空调位等 |

### 3.3 故障-方案映射（核心10种场景）

| 漏点位置 | 故障类型 | 严重程度 | 方案名称 | 工期 | 价格区间 |
|----------|----------|----------|----------|------|----------|
| 卫生间 | 水渍 | 轻微 | 局部防水涂层 | 0.5天 | 200-500元 |
| 卫生间 | 水渍 | 普通 | 卫生间重做防水 | 2-3天 | 1500-3000元 |
| 卫生间 | 裂缝 | 轻微 | 裂缝灌浆处理 | 0.5天 | 300-600元 |
| 卫生间 | 裂缝 | 普通 | 裂缝修复+防水 | 1-2天 | 800-1500元 |
| 卫生间 | 渗漏 | 严重 | 注浆防水+整体重做 | 3-5天 | 3000-6000元 |
| 屋顶 | 裂缝 | 普通 | 裂缝修补+防水卷材 | 1-2天 | 1000-2000元 |
| 屋顶 | 渗漏 | 严重 | 整体防水+隔热层 | 5-7天 | 5000-10000元 |
| 外墙 | 泛碱 | 普通 | 墙面清理+防水涂料 | 2-3天 | 1500-2500元 |
| 阳台 | 潮湿 | 轻微 | 地面防水涂层 | 0.5天 | 300-600元 |
| 厨房 | 渗漏 | 普通 | 管道检修+局部防水 | 1天 | 500-1000元 |

### 3.4 故障诊断展示

用户看到的内容：
- **诊断结果**：根据图片分析，您的漏水问题为「卫生间墙角水渍 + 轻微」
- **初步说明**：疑似防水层老化，建议尽快处理避免恶化
- **预估价格**：200-500 元（仅供参考）

### 3.5 每日任务

| 天数 | 任务 |
|------|------|
| 第1天 | 专家确认故障-方案映射表（10个核心场景） |
| 第2天 | 数据库设计、知识库后端 API 开发 |
| 第3天 | 前端集成、联调测试 |

## 四、子项目3：智能勘察预约（3天）— 项目最终一步

### 4.1 功能目标
引导客户下单预约工程师上门勘察，生成工单。

### 4.2 用户流程

```
故障诊断结果展示
     │
     ▼
┌─────────────────┐
│  了解初步报价    │ ◄── 显示预估价格区间
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  预约上门勘察   │ ◄── 引导客户下单
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  填写联系信息   │ ◄── 姓名、电话、地址
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  生成工单       │ ◄── 推送给工程师
└─────────────────┘
```

### 4.3 核心功能

| 功能 | 说明 |
|------|------|
| 初步报价展示 | 显示预估价格区间，让客户心里有底 |
| 勘察预约按钮 | 引导语：「预约工程师上门勘察，确认最终方案」 |
| 信息填写 | 姓名 + 电话 + 详细地址 + 方便上门时间 |
| 工单生成 | 创建工单，推送给后台工程师 |

### 4.4 引导话术

```
根据您上传的图片分析，漏水问题为：卫生间墙角水渍（轻微）
预估价格：200-500元

💡 准确价格需要工程师上门勘察后确认
▶ 预约工程师上门勘察（免费）
```

### 4.5 每日任务

| 天数 | 任务 |
|------|------|
| 第1天 | 预约流程设计、引导话术确认 |
| 第2天 | 后端工单 API 开发 |
| 第3天 | 前端预约页面、提交联调 |
