1. 项目概述

🎯 产品定位

用户拍照即可获得防水故障评估、施工方案推荐和价格区间，快速生成工单的 H5 应用，嵌入微信/抖音小程序 WebView。

1.1 核心价值

  • 用户侧：拍照即可获取专业防水评估和报价，透明化价格
  • 业务侧：自动化初步诊断，减少客服成本，提高转化率
  • 技术侧：AI 图像识别 + 规则引擎，智能化匹配施工方案

1.2 目标用户

用户类型 使用场景 核心需求
家庭用户 发现家中漏水，需要快速评估和报价 快速、准确、价格透明
物业管理人员 处理小区住户报修 批量处理、专业评估
师傅/客服 接收和处理工单 工单信息完整、便于跟进

3. 产品需求文档 (PRD)

3.1 核心功能详细说明

漏点位置选择（7 个）

卫生间 | 厨房 | 阳台 | 屋顶 | 外墙 | 飘窗 | 其他

AI 识别病害类型（6 种）

病害类型 说明 典型场景
水渍 墙面/地面水渍印记 卫生间墙角发霉变黑
裂缝 墙面/地面裂缝 屋顶裂纹、墙体开裂
潮湿 局部潮湿、凝水 地下室墙面渗水
泛碱 白色碱化结晶 瓷砖缝、外墙起白硝
渗漏 明显漏水痕迹 水珠滴落、明显水流
发霉 霉菌斑块 黑色/绿色霉斑

严重程度判定

等级 判定依据
轻微 局部小面积（<10cm²）、早期症状、不影响结构
普通 明显可见、范围适中（10-50cm²）、需要处理
严重 大面积（>50cm²）、已影响结构、需紧急处理

卫生间 水渍 普通 卫生间重做防水 2-3 天 1500-3000 元
卫生间 裂缝 轻微 裂缝灌浆处理 0.5 天 300-600 元
卫生间 渗漏 严重 注浆防水 + 整体重做 3-5 天 3000-6000 元
屋顶 裂缝 普通 裂缝修补 + 防水卷材 1-2 天 1000-2000 元
屋顶 渗漏 严重 整体防水 + 隔热层 5-7 天 5000-10000 元

6.3 部署步骤

步骤 1：购买云服务器和域名

# 推荐配置
云服务器：2 核 CPU, 4GB 内存，50GB 系统盘
操作系统：Ubuntu 22.04 LTS 或 CentOS 7+
带宽：3-5Mbps（初期）

步骤 2：配置域名解析

# 在域名服务商添加 A 记录
类型：A
主机记录：quote（或@）
记录值：你的服务器公网 IP
TTL：600

步骤 3：安装 SSL 证书（HTTPS）

# 使用 Let's Encrypt 免费证书
sudo apt install certbot python3-certbot-nginx
sudo certbot --nginx -d quote.yourdomain.com

步骤 4：部署后端服务

# 安装 Python 和依赖
sudo apt install python3-pip python3-venv
pip3 install fastapi uvicorn pymysql

# 配置环境变量
export DATABASE_URL="mysql+pymysql://user:pass@localhost/waterproof_quote"
export OSS_ACCESS_KEY="your_key"
export AI_API_KEY="your_minimax_key"

# 启动服务（生产环境用 Gunicorn）
gunicorn -w 4 -k uvicorn.workers.UvicornWorker app.main:app --bind 0.0.0.0:8000

步骤 5：配置 Nginx 反向代理

server {
    listen 443 ssl;
    server_name quote.yourdomain.com;
    
    ssl_certificate /etc/letsencrypt/live/quote.yourdomain.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/quote.yourdomain.com/privkey.pem;
    
    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
    
    # 前端静态文件
    location /static {
        alias /var/www/waterproof-quote-h5/dist;
    }
}

步骤 6：部署前端 H5

# 构建前端
npm install
npm run build

# 将 dist 目录上传到服务器
scp -r dist/* user@server:/var/www/waterproof-quote-h5/

6.4 快速部署方案（推荐新手）

🚀 使用 Docker Compose 一键部署

version: '3.8'
services:
  backend:
    build: ./backend
    ports:
      - "8000:8000"
    environment:
      - DATABASE_URL=mysql+pymysql://db:3306/waterproof_quote
      - OSS_ACCESS_KEY=${OSS_ACCESS_KEY}
      - AI_API_KEY=${AI_API_KEY}
    depends_on:
      - db
    restart: always

  db:
    image: mysql:8.0
    environment:
      MYSQL_ROOT_PASSWORD: ${DB_PASSWORD}

  frontend:
    image: nginx:alpine

      - ./nginx.conf:/etc/nginx/conf.d/default.conf
    ports:
      - "80:80"
      - "443:443"
    depends_on:
      - backend
    restart: always
