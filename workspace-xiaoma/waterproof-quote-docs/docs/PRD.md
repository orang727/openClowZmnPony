# 防水补漏智能报价器 - 产品需求文档 (PRD)

**文档版本**：v1.0  
**创建日期**：2026-03-05  
**产品名称**：防水补漏智能报价器  
**产品形态**：H5 页面（嵌入小程序 WebView）

---

## 一、产品概述

### 1.1 产品定位

| 项目 | 说明 |
|------|------|
| **产品名称** | 防水补漏智能报价器 |
| **产品形态** | H5 页面（嵌入微信/抖音小程序 WebView） |
| **核心价值** | 用户拍照即可获得防水故障评估、施工方案推荐和价格区间，快速生成工单 |
| **目标用户** | 有防水补漏需求的家庭用户 |

### 1.2 产品功能清单

| 序号 | 功能模块 | 功能描述 | 优先级 |
|------|----------|----------|--------|
| 1 | 漏点位置选择 | 用户选择漏水位置（卫生间/厨房/阳台/屋顶/外墙/飘窗/其他） | P0 |
| 2 | 建筑信息填写 | 楼层、房龄、面积段（可选） | P1 |
| 3 | 问题描述 | 用户补充文字描述（可选） | P1 |
| 4 | 现场图片上传 | 上传 1-5 张现场照片 | P0 |
| 5 | AI 图像分析 | 根据图片识别漏水严重程度（轻微/普通/严重） | P0 |
| 6 | 智能方案推荐 | 根据严重程度匹配施工方案 | P0 |
| 7 | 价格区间显示 | 显示每个方案的报价区间 | P0 |
| 8 | 确认下单 | 用户选择方案后生成工单 | P0 |
| 9 | 工单推送 | 推送到工单系统 | P0 |
| 10 | 小程序回调 | 工单完成后回调小程序 | P1 |

## 三、功能详细说明

### 3.1 漏点位置选择

- **组件**：Picker 选择器
- **选项**：
  - 卫生间
  - 厨房
  - 阳台
  - 屋顶
  - 外墙
  - 飘窗
  - 其他
- **必填**：是
- **交互**：点击弹出底部选择器，选择后回显

### 3.2 建筑信息（可选）

| 字段 | 选项 | 说明 |
|------|------|------|
| **楼层** | 低层 / 中层 / 高层 | 影响施工难度和价格 |
| **房龄** | 0-5年 / 5-10年 / 10年以上 | 影响防水层老化程度评估 |
| **面积段** | <50㎡ / 50-100㎡ / >100㎡ | 影响施工面积和价格 |

### 3.3 问题描述

- **组件**：多行文本框
- **字数限制**：200 字
- **占位符**：例如：阴雨天墙角会渗水，有水渍印记，地面偶尔有积水...

### 3.4 现场图片上传

| 规格 | 说明 |
|------|------|
| **数量** | 1-5 张 |
| **单张大小** | 最大 8MB |
| **格式** | 支持 jpg、png、jpeg |
| **压缩** | 前端压缩后再上传 |
| **必填** | 是（至少 1 张） |

**拍摄建议**：
- 1 张整体环境
- 1-3 张近距离细节（漏水点、水渍、裂缝等）
- 避免背光和过暗

### 3.5 AI 图像分析

- **输入**：用户上传的图片 + 位置信息 + 文字描述
- **输出**：
  - 严重程度：`light`（轻微）/`normal`（普通）/`severe`（严重）
  - 严重程度标签：轻微 / 普通 / 严重
  - 分析说明：文字描述（如：疑似防水层老化，局部开裂）
- **实现方式**：调用 AI 模型 API（支持配置 minimax/OpenAI/自研）

### 3.6 智能方案推荐

- **匹配逻辑**：严重程度 + 漏点位置 + 建筑信息 → 施工方案列表
- **每个方案包含**：
  - 方案名称
  - 方案描述
  - 标签（经济型/标准型/加强型）
  - 预计工期
  - 价格区间（min - max）

### 3.7 价格区间显示

- **展示形式**：￥800 - ￥1200
- **说明**：报价为预估区间，现场勘查后会给出最终报价

### 3.8 确认下单

- **前提**：用户已选择一个方案
- **操作**：点击「生成工单」按钮
- **结果**：创建工单，返回工单号

### 3.9 工单推送

- **内容**：用户信息、漏水位置、严重程度、选择方案、报价区间、图片
- **推送目标**：工单系统

### 3.10 小程序回调

- **时机**：工单创建成功后
- **方式**：调用 `wx.miniProgram.navigateBack` 或 `postMessage`
- **参数**：工单号 `order_id`

## 五、非功能性需求

### 5.1 性能要求

| 指标 | 要求 |
|------|------|
| 页面加载时间 | < 3 秒 |
| 图片上传响应 | < 5 秒（5MB 以内） |
| 估价响应时间 | < 10 秒（含 AI 分析） |

### 5.2 兼容性

- 微信小程序 WebView
- 抖音小程序 WebView
- iOS / Android 系统

### 5.3 安全要求

- 用户信息加密传输
- 图片存储需要备份
- API 接口需要鉴权

## 六、后续迭代规划

### 6.1 图片识别防水故障（子项目1）

- 基于图像识别技术自动识别漏水类型
- 识别结果：水渍、裂缝、潮湿、泛碱等

### 6.2 智能故障匹配（子项目2）

- 根据故障类型匹配专业解决方案
- 建立故障-方案映射知识库

### 6.3 智能方案推荐匹配（子项目3）

- 基于历史案例智能推荐最优方案
- 支持个性化推荐算法
