# 产品分类设计 Skill

## 概述

用于设计前台产品与后台产品的分类体系，适用于服务类业务（如防水、环氧地坪等）。

---

## 产品层级结构

```
服务分类 → 一级分类 → 前台产品 → 关联后台产品
                                    ↓
                              后台产品 → 关联服务项
                                    ↓
                              服务项（最小报价单位）
```

### 业务逻辑

1. **前台产品**：客户选择的入口，简化为主
2. **后台产品**：工程师上门后根据现场情况选择
3. **服务项**：最小报价单位，有价格、单价、单位
4. **个性化报价**：通过服务项自由组合实现

---

## 前台产品设计原则

### 设计要点

| 原则 | 说明 |
|------|------|
| **简化** | 前台产品宜简，客户对工艺不专业 |
| **入口少** | 1个入口优于多个，让客户简单选择 |
| **工程师选择** | 具体工艺由工程师上门后选择 |

### 示例

| 业务 | 前台产品设计 |
|------|--------------|
| 防水补漏 | "防水补漏"（1个入口） |
| 环氧地坪 | "环氧地坪"（1个入口） |
| 墙面翻新 | "墙面翻新"（可按类型分） |

---

## 后台产品设计原则

### 设计要点

| 原则 | 说明 |
|------|------|
| **按工艺区分** | 不同工艺类型作为后台产品 |
| **可选择** | 工程师根据现场情况选择 |
| **关联服务项** | 每个后台产品关联具体服务项 |

### 示例

| 前台产品 | 后台产品（按工艺分） |
|----------|---------------------|
| 环氧地坪 | 薄涂型、砂浆型、自流平型、彩砂型、停车场 |

---

## 服务项设计

### 服务项组成

每个服务项包含：
- **名称**：服务项名称
- **材料费**：材料用量 × 单价
- **施工费**：人工成本
- **单位**：元/㎡、元/米、元/个等

### 定价规则

参考 `pricing` skill 的定价公式：
```
售价 = (材料成本 + 施工成本) / 0.73
```

## 产品分类模板

### Excel 输出格式

#### Sheet 1: 前台产品

| 字段 | 说明 |
|------|------|
| 服务分类 | 一级分类（如：墙面） |
| 一级分类 | 业务大类（如：环氧地坪） |
| 二级分类 | 可选，细分子类 |
| 前台产品 | 客户可见的产品名 |
| 关联后台产品 | 可关联的后台产品列表 |

#### Sheet 2: 后台产品

| 字段 | 说明 |
|------|------|
| 后台产品名称 | 工程师可见 |
| 工艺类型 | 区分不同工艺 |
| 建议报价 | 参考价格 |
| 说明 | 产品描述 |

#### Sheet 3: 服务项

| 字段 | 说明 |
|------|------|
| 后台产品 | 所属后台产品 |
| 服务项 | 最小服务单位 |
| 材料费 | 材料成本 |
| 施工费 | 人工成本 |
| 单价 | 售价 |
| 单位 | 元/㎡、元/米等 |

## 使用示例：环氧地坪

### 前台设计

| 服务分类 | 一级分类 | 前台产品 | 关联后台产品 |
|----------|----------|----------|--------------|
| 墙面 | 环氧地坪 | 环氧地坪 | 环氧地坪 |

### 后台设计

| 后台产品 | 工艺类型 | 建议报价 |
|----------|----------|----------|
| 薄涂型环氧地坪 | 薄涂型 | 35元/㎡起 |
| 砂浆型环氧地坪 | 砂浆型 | 55元/㎡起 |
| 自流平型环氧地坪 | 自流平型 | 80元/㎡起 |
| 彩砂型环氧地坪 | 彩砂型 | 150元/㎡起 |
| 停车场地坪 | 停车场 | - |

### 服务项设计

每个后台产品关联多个服务项：
- 地面基层处理
- 底漆
- 中涂
- 面漆
- 人工费
