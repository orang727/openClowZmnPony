2. 用 GitHub/GitLab/Bitbucket 账号登录（或注册）
3. 点击 **"Add New Project"**
4. 选择 **"Continue with Third Party"** → 跳过 Git，直接拖拽上传
5. 把以下文件夹拖进去：
   ```
   C:\Users\pony\clawd-xiaoma\waterproof-quote-project
   ```
6. 等待部署完成（约 30 秒）
7. 获得永久链接：`https://your-project.vercel.app`

### 优点：
- ✅ 完全免费
- ✅ 自动 HTTPS
- ✅ 永久有效
- ✅ 支持自定义域名

## 方案 2：Netlify Drop（无需账号，临时）

### 步骤：
1. 访问 **https://app.netlify.com/drop**
2. 直接把文件夹拖到页面
3. 获得临时链接（无需注册）

### 升级永久：
- 注册账号后绑定链接，即可永久有效

---

## 方案 3：GitHub Pages（永久，需要 GitHub 账号）

### 步骤：
1. 访问 **https://github.com/new**
2. 创建新仓库（例如：`waterproof-quote`）
3. 上传 HTML 文件
4. 进入仓库 **Settings** → **Pages**
5. 选择 **main branch** → 保存
6. 获得链接：`https://yourusername.github.io/waterproof-quote/`
