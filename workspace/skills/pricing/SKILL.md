# 商业产品定价计算器 Skill

## 概述

这是一个用于商业产品定价的 Excel 计算工具，基于成本加成法，适用于服务类产品的报价计算。

## 定价公式

### 核心公式

```
售价 = (材料成本 + 施工成本) / 0.73
```

其中 0.73 = 1 - 19% - 8%
- 19% = 费用率 (0.49% + 8% + 8% + 3%)
- 8% = 净利润率

### 详细公式

```
成本 = 材料成本 + 施工成本

费用 = 售价 × 19%
  = 售价 × (0.49% + 8% + 8% + 3%)
  = 售价 × 0.19

利润 = 售价 × 8%

售价 = 成本 + 费用 + 利润
售价 = (材料成本 + 施工成本) + 售价 × 0.19 + 售价 × 0.08
售价 = (材料成本 + 施工成本) / 0.73
```

### 毛利和净利计算

```
预估毛利率(%) = (售价 - 材料成本 - 施工成本) / 售价 × 100
预估净利率(%) = 8% (锁定)
```

## 边界条件

### 1. 地面基层处理
- 统一价格：7元/㎡（不分材料差异）
- 原因：客户感知为一个工艺

### 2. 施工费分摊
- 按服务项数量平均分摊到每个服务项
- 确保每个服务项都有施工费

### 3. 最低施工面积
- 计算公式：最低施工面积 = 800元 / 人工费单价
- 保证工人上门1天的基本施工费（2人 × 400元 = 800元）

| 工艺类型 | 人工费(元/㎡) | 最低面积(㎡) | 最低施工费(元) |
|----------|---------------|--------------|----------------|
| 薄涂型 | 12 | 67 | 800 |
| 砂浆型 | 15 | 54 | 800 |
| 自流平型 | 18 | 45 | 800 |
| 彩砂型 | 20 | 40 | 800 |
| 停车场 | 15 | 54 | 800 |

## Excel 输出格式

### Sheet 1: 服务项定价

| 字段 | 说明 |
|------|------|
| 后台产品 | 产品名称 |
| 服务项名称 | 具体服务项 |
| 服务项单价(元/㎡) | 四舍五入取整 |
| 材料费(元/㎡) | 材料成本 |
| 施工费(元/㎡) | 分摊后的施工成本 |
| 预估毛利(%) | 毛利率 |
| 预估净利(%) | 固定8% |

| 字段 | 说明 |
|------|------|

| 施工人数 | 2人 |
| 施工天数 | 1天 |
| 人工成本(元/人/天) | 400元 |
| 备注 | 说明 |

## 使用方法

### 输入数据格式 (Python)

```python
# 配置示例
configs = {
    '薄涂型': {
        'items': [
            ('地面基层处理', 0.2, 8),   # (名称, 用量, 单价)
            ('环氧渗透底漆', 0.15, 25),
            ('环氧面漆', 0.4, 30)
        ],
        'labor': 12,    # 人工费(元/㎡)
        'min_area': 67   # 最低施工面积
    },
    # ... 其他工艺类型
}

# 统一价格（不按材料成本计算）
ground_price = 7  # 地面基层处理统一价
```

### 执行计算

```bash
python pricing_calc.py
```

输出：`环氧地坪定价v5.xlsx`
